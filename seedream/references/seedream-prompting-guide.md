# Seedream 提示词指南

基于 [火山方舟官方指南](https://www.volcengine.com/docs/82379/1829186) 和 [Seedream Prompt Guide](https://www.seedream4.net/prompt-guide) 整理。

## Prompt 公式

### 文字生图 (Text-to-Image)

```
[主体] + [动作/行为] + [场景/环境] + [构图] + [风格/光影]
```

模型理解优先级：**主体 → 风格 → 构图 → 光线 → 相机/技术细节**

**示例**：
> 充满活力的特写编辑肖像，模特眼神犀利，头戴雕塑感帽子，色彩拼接丰富，具有 Vogue 杂志封面的美学风格，中画幅拍摄，工作室灯光强烈。

> A girl in elegant clothing, holding a parasol, walking down a tree-lined avenue, Monet oil painting style, warm golden hour light.

### 图生图编辑 (Image Editing)

使用 **动作 + 对象 + 属性** 结构：

| 动作 | 说明 | 示例 |
|------|------|------|
| 替换 (Replace) | 改变某物的属性 | 将骑士的头盔改为金色 |
| 移除 (Remove) | 删除画面元素 | 移除背景中的行人 |
| 添加 (Add) | 增加新元素 | 在天空中添加一轮明月 |
| 参考 (Reference) | 融合参考图特征 | 将图1的人物放入图2的场景中 |

### 多图融合 (Multi-Image)

指定每张参考图的角色：

```
将图1的人物穿上图2的服装，放在图3的场景中，使用图4的色彩风格。
```

### 组图生成 (Sequential)

使用关键词触发：「一组」「一系列」「连续」「四季」等

```
生成一组共4张连贯插画，核心为同一庭院一角的四季变迁，统一水彩风格。
```

## 高级技巧

### 风格关键词

| 类型 | 关键词示例 |
|------|-----------|
| 摄影 | 电影调色、胶片质感、高对比度、浅景深 f/1.8 |
| 插画 | 扁平化矢量、日系水彩、像素风、手绘涂鸦 |
| 3D | C4D 渲染、等距视角、玻璃材质、金属反射 |
| 国风 | 工笔画、泼墨山水、敦煌壁画风、青花瓷纹样 |
| 商业 | 产品摄影白底、杂志封面版式、简约海报设计 |

### 灯光与氛围

- 三点柔光箱布局（工作室）
- 黄金时段逆光（户外暖调）
- 明暗对比 Chiaroscuro（戏剧感）
- 霓虹灯光（赛博朋克）
- 柔和漫反射光（治愈系）

### 构图控制

- 可在 prompt 中直接描述宽高比：「竖版海报」「横版壁纸」「方形头像」
- 构图术语：中心对称、三分法、引导线、框架构图、低角度仰拍
- 镜头：广角全景、微距特写、鱼眼、航拍俯瞰

### 文字渲染

- 用引号包裹目标文字：`写着"新年快乐"的红色横幅`
- 指定字体风格：`粗体黑体` `手写体` `哥特式字母`
- 支持中英文混排

### 色彩控制

- 指定色调：`暖色调` `冷蓝色调` `莫兰迪色系`
- 配色方案：`红金配色的喜庆氛围` `黑白高对比`
- 材质色彩：`拉丝金属质感` `磨砂玻璃` `丝绒红`

## 最佳实践

1. **长度适中**：30-100 词，3-5 个关键形容词，避免堆砌
2. **积极描述**：说"空旷的街道"而非"没有车的街道"
3. **专业术语用原文**：摄影/艺术术语保留英文可能效果更好
4. **定义应用场景**：如"PPT 封面背景""社交媒体方形配图"
5. **迭代优化**：一次只修改一个变量，对比效果
6. **中英文均可**：Seedream 对中文理解能力强，日常用中文即可

## 联网搜索生图（仅 5.0 lite）

可利用实时信息生成图片：

```
制作一张北京今日天气预报图，扁平化信息图表风格，包含温度和穿搭建议。
```

配合 `tools: [{"type": "web_search"}]` 参数使用。
